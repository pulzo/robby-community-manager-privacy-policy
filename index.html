POLÍTICA DE PRIVACIDAD

Última actualización: 10 diciembre 2025

1. Introducción

Esta Política de Privacidad explica cómo Robby Community Manager procesa información relacionada con la publicación automática de contenido desde Pulzo.com a Facebook. La aplicación respeta la privacidad de los usuarios y cumple con las políticas de la API de Facebook y las normativas aplicables.

2. Información que procesa la aplicación

Robby Community Manager no recopila ni almacena datos personales de usuarios.

La aplicación únicamente procesa los siguientes datos técnicos:

Sobre los contenidos publicados:

  • Título del artículo
  • URL pública
  • Imagen asociada (si aplica)
  • Fecha de publicación

Información técnica necesaria para la integración:

  • Token de acceso de Facebook (Page Access Token)
  • ID de la página de Facebook donde se publicará el contenido

Esta información se utiliza exclusivamente para realizar la acción de publicación.

3. ¿Cómo se obtiene la información?

  • De la base de contenido de Pulzo.com y sus aliados.
  • De la API oficial de Facebook para publicar contenido.
  • No se recopila información de usuarios finales de Facebook.

4. Uso de la información

La información procesada tiene un solo propósito: Publicar artículos periodísticos en páginas de Facebook autorizadas.

No se realiza:

  • Perfilamiento de usuarios.
  • Recolección de cookies.
  • Venta o cesión de datos.
  • Análisis de comportamiento de usuarios finales.

5. Permisos de Facebook

Robby Community Manager solicita únicamente permisos necesarios para publicar contenido en nombre de una página:

  • pages_manage_posts
  • pages_read_engagement

No se solicitan permisos adicionales.

6. Retención y almacenamiento

  • Los tokens de acceso se almacenan de forma segura.
  • No se almacenan copias de información personal.
  • No se guardan métricas de usuarios.

Los registros técnicos de publicación pueden conservarse con fines internos de auditoría.

7. Seguridad

La aplicación utiliza prácticas estándar para proteger tokens de acceso y solicitudes hacia Facebook:

  • HTTPS para todas las comunicaciones.
  • Tokens cifrados.
  • Control de acceso interno.

8. Derechos del usuario

Dado que Robby Community Manager no recopila datos personales de usuarios, no aplica la solicitud de eliminación, acceso o portabilidad de datos personales.

9. Cambios en la Política

Podemos actualizar esta Política para reflejar mejoras en el sistema o cambios normativos. La fecha de actualización se mostrará al inicio del documento.

Transparencia editorial

Robby Community Manager respeta las políticas de publicación de Facebook y las prácticas editoriales de Pulzo, garantizando que el contenido distribuido:

  • se origina en fuentes verificadas,
  • mantiene integridad periodística,
  • no altera el artículo original,
  • solo publica contenido autorizado por Pulzo.

Estado actual del proyecto

Robby Community Manager está en fase activa de desarrollo y mejora continua. El proyecto forma parte del ecosistema interno de herramientas de Pulzo para optimizar:

  • flujo de publicación,
  • gestión editorial,
  • presencia en redes sociales.
