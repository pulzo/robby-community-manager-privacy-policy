POLÍTICA DE PRIVACIDAD – ROBBY COMMUNITY MANAGER

Última actualización: 10 diciembre 2025

1. Introducción

Esta Política de Privacidad explica cómo Robby Community Manager procesa información relacionada con la publicación automática de contenido desde Pulzo.com y sus aliados a páginas de Facebook.

La aplicación es desarrollada y operada por Pulzo.com (R360 S.A.S.) y respeta la privacidad de los usuarios, cumple con las políticas de la API de Facebook y con las normativas de protección de datos aplicables.

Robby Community Manager puede utilizarse tanto en páginas propias de Pulzo como en páginas de aliados autorizados.

2. Información que procesa la aplicación

Robby Community Manager no recopila ni almacena datos personales de usuarios finales de Facebook.

La aplicación únicamente procesa la siguiente información técnica necesaria para la publicación de contenido:

Sobre los contenidos publicados:

  • Título del artículo.
  • URL pública del artículo.
  • Imagen asociada (si aplica).
  • Fecha de publicación.

Información técnica para la integración con Facebook:

  • Token de acceso de Facebook (Page Access Token) de la página autorizada.
  • ID de la página de Facebook donde se publicará el contenido.

Esta información se utiliza exclusivamente para realizar la acción de publicación de artículos periodísticos en páginas de Facebook autorizadas.

3. ¿Cómo se obtiene la información?

  • La base de contenido de Pulzo.com y sus aliados.
  • La API oficial de Facebook para la publicación de contenido en páginas.

Robby Community Manager no recopila información de usuarios finales de Facebook, como nombres, correos electrónicos o perfiles personales.

4. Uso de la información

La información procesada tiene un único propósito:

Publicar artículos periodísticos en páginas de Facebook autorizadas por Pulzo o sus aliados.

No se realiza:

  • Perfilamiento de usuarios.
  • Recolección de cookies.
  • Venta o cesión de datos a terceros.
  • Análisis de comportamiento de usuarios finales de Facebook.

5. Permisos de Facebook

Robby Community Manager solicita únicamente los permisos estrictamente necesarios para publicar contenido en nombre de una página de Facebook:

  • pages_manage_posts
  • pages_read_engagement

No se solicitan permisos adicionales para acceder a información personal de usuarios finales.

6. Retención y almacenamiento

  • Los tokens de acceso se almacenan de forma segura y solo se utilizan para publicar en las páginas autorizadas.
  • No se almacenan copias de información personal de usuarios finales.
  • No se guardan métricas de usuarios ni estadísticas de comportamiento individual.
  • Los registros técnicos de publicación (por ejemplo, fecha, hora y resultado) pueden conservarse con fines internos de auditoría, monitoreo y mejora del servicio.

Si se revoca el acceso de la aplicación desde la configuración de Facebook, los tokens dejan de ser válidos y la aplicación ya no podrá publicar en esa página.

7. Seguridad

Robby Community Manager utiliza prácticas estándar de la industria para proteger los tokens y las comunicaciones:

  • Uso de HTTPS para todas las comunicaciones.
  • Tokens cifrados y protegidos.
  • Controles internos de acceso al sistema.

Pulzo revisa y mejora periódicamente sus medidas de seguridad para reducir riesgos de acceso no autorizado o uso indebido.

8. Derechos relacionados con los datos

Dado que Robby Community Manager no recopila datos personales de usuarios finales de Facebook, no aplica el ejercicio de derechos como acceso, rectificación o portabilidad.

No obstante, si eres representante, administrador o aliado de una página que utiliza la aplicación y consideras que tus datos personales han sido tratados, puedes escribirnos para:

  • Solicitar información sobre el tratamiento de tus datos.
  • Solicitar actualización, corrección o eliminación de información que te identifique, si aplica.

Cada solicitud será evaluada conforme a la normativa de protección de datos aplicable.

9. Cambios en la Política

Podemos actualizar esta Política de Privacidad para reflejar mejoras del sistema, cambios tecnológicos o ajustes normativos. La fecha de última actualización se mostrará al inicio del documento. Recomendamos revisar esta página periódicamente.

Transparencia editorial

Robby Community Manager respeta las políticas de publicación de Facebook y las prácticas editoriales de Pulzo, garantizando que el contenido distribuido:

  • Se origina en fuentes verificadas,
  • Mantiene integridad periodística,
  • No altera el artículo original,
  • Solo publica contenido autorizado por Pulzo y/o sus aliados.

Estado actual del proyecto

Robby Community Manager se encuentra en fase activa de desarrollo y mejora continua. El proyecto contribuye a optimizar:

  • El flujo de publicación,
  • La gestión editorial,
  • Y la presencia en redes sociales.
